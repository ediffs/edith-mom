# Edith's Site!

## Overview

This is my personal website and portfolio, built with [React](https://react.dev/), [Next.js](https://nextjs.org), and [Tailwind CSS](https://tailwindcss.com/). 

The project was bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Contributing

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the site.

You can start editing the page by modifying `page.tsx` in each page's folder. The page auto-updates as you edit the file.

## Learn More

The main resources I used while creating this website were (in order of usefulness):
* [Tailwind CSS Docs](https://tailwindcss.com/docs/installation/using-vite) - The only one of my main tools with solid documentation on its own.
* [GeeksForGeeks](https://www.geeksforgeeks.org) - Useful for their React tutorials.
* [React Quick Start](https://react.dev/learn) - A good way to familiarize yourself with React, although I'd recommend just building something as soon as you feel ready.
* [Next.js Docs](https://nextjs.org/docs) - Not the best documentation but has what you might need for certain tasks.
